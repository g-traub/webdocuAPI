# API pour le web documentaire

## Installer
* Dans la console :
~~~~
npm i
~~~~

* Créer un fichier .env à la racine du projet, au format :
~~~~
PORT=3001
MONGO_USER=user
MONGO_PSWD=user
~~~~

* Dans la console :
~~~~
npm start
~~~~
